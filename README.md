# ATONEMENT - Interactive Ethics Short Film

> *"In a game of survival, loyalty is the first casualty. Who will you sacrifice?"*

**Atonement** is an interactive web-based short film experience inspired by the Netflix UI. It explores deep ethical themes through a psychological thriller narrative, placing the viewer in the driver's seat to make difficult moral choices.

## üé¨ About the Film

Four successful high school friends‚ÄîJoaquin, Nigel, Chloe, and Bea‚Äîare lured to a mysterious reunion at "The Loft." Trapped in a secluded place, they are forced to confront a dark secret from their past: the bullying and death of their classmate, Abby.

As their carefully constructed lives crumble into madness, they must face the terrifying reality that death is easy, but true **atonement** is hell.

### Key Ethical Themes
* **Moral Dilemma:** Utilitarianism vs. Deontology (Sacrifice one vs. Duty to all).
* **Human Dignity:** The consequences of dehumanization and bullying.
* **Accountability:** Retributive justice and the cost of silence.

## ‚ú® Features

* **Netflix-Inspired UI:** A fully responsive, immersive interface with a dynamic navbar, hero section, and modal popups.
* **Interactive Storytelling:** Viewers make crucial decisions that alter the ending of the film.
* **Suspense Timer:** A "Decide Their Fate" screen with a countdown timer that forces a default choice if the viewer hesitates.
* **Seamless Video Player:** Custom-configured YouTube player integration for a cinematic experience.
* **Responsive Design:** Optimized for Desktop, Tablet, and Mobile devices.
* **Cast & Crew Showcase:** Dedicated modal to highlight the actors and production team.

## üöÄ Getting Started

Follow these steps to run the project locally.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/BagunasJohnrey/atonement.git
    cd atonement
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```

4.  Open your browser and navigate to `http://localhost:5173`.
